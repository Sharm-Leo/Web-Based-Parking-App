# 🚗 Web-Based Parking Management System

A full-stack web application developed to simplify parking space management through an intuitive interface for users and administrators. The system enables vehicle registration, parking allocation, occupancy tracking, and administrative management while demonstrating full-stack web development concepts.

## Features

- User Authentication
- Vehicle Registration
- Parking Slot Management
- Real-time Availability
- Booking & Allocation
- Administrative Dashboard
- Database Integration
- Responsive Interface

## Tech Stack

Frontend
- HTML5
- CSS3
- JavaScript
- Bootstrap

Backend
- SQLAlchemy
- Python
